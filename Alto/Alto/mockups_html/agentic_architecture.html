3 — Claude Agent Loop (claude-sonnet-4-6)
CLAUDE SONNET 4.6
Alto Coaching Agent
Reasons step-by-step. Calls tools. Stops when plan is complete and saved.
1
Read recovery signals
Fetch sleep, HRV, active energy from HealthKit. Compute readiness score.
fetch_healthkit()
2
Check environment
Fetch weather forecast. Flag rain, heat, or extreme conditions.
fetch_weather()
3
Load user context
Goal, training phase, health conditions, preferences, cycle phase.
get_user_profile()
4
Check what's already planned
User said "Pilates 45 min". Read calendar bookings. Don't double-schedule.
get_planned_activities()
5
Evaluate calorie gap
Pilates 45 min ≈ 135 kcal. Goal is 400 kcal. Gap = 265 kcal. Decide whether to suggest add-on.
evaluate_calorie_gap() → reasoning: "Add a 30 min walk to close the gap gently"
6
Pivot check
Readiness = 38/100 (high soreness). Override Pilates → Rest + light walk. Flag pivot.
→ reasoning: "Readiness too low for any structured session today"
7
Build & save final plan
Write structured plan with activities, kcal, RPE, coach note, pivot flag to UserStore.
save_plan() ⏹ STOP — plan complete

6 — Post-Workout Feedback Loop
Effort Gap Check
After workout: actual kcal burned vs planned. Too low → compassionate re-engagement. Too high vs easy plan → mandatory recovery.
RPE Debrief
User rates perceived effort 1–10. Agent compares to target RPE. Adjusts tomorrow's plan intensity accordingly.
Streak & Momentum
Agent tracks consistency. After 3 days skipped → gentle re-entry plan. After 7 days consistent → unlock next phase early.
